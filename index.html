Props

PropTypeDefaultDescription
v-modelString Array

Array of of tag Names

Example v-model variable should contain a String array similar to this. : ['VueJS','Vite']

UseTwoBackspaceDeleteBooleantrue

If you want to delete previous tag on two Backspaces rather than one.

Example: :UseTwoBackspaceDelete="true"

classString

Class to style the outer most UL element of Tags component

Example : flex flex-wrap gap-2

class-delete-iconString

Classes you want on Cross Icon on each tag

Example : h-5 rounded hover:bg-red-600 w-5 cursor-pointer
before-delete-tag-classString

The Classes mentioned here are applied to the tag which is about to be deleted when 1 backspace is pressed.

Example : before-delete-tag-class="pl-4 pr-2 py-1 rounded bg-red-100 dark:bg-red-900 flex items-center gap-2"
tag-classString

CSS Class to be applied on the tag

Example : tag-class="pl-4 pr-2 py-1 rounded bg-slate-100 dark:bg-slate-900 flex items-center gap-2"
tabsContentClassString

CSS Classes for the wrapper on the area where tab contents are displayed.

Example : input-class="bg-gray-50 border border-gray-300 text-gray-900 text-sm rounded-lg focus:ring-blue-500 focus:border-blue-500 block w-full px-4 py-1.5 dark:bg-gray-700 dark:border-gray-600 dark:placeholder-gray-400 dark:text-white dark:focus:ring-blue-500 dark:focus:border-blue-500" />
